# VAD モデル配置用

## silero_vad_16k_op15.onnx

このフォルダに **silero_vad_16k_op15.onnx** を置いてください。

### ダウンロード

- **URL**: https://huggingface.co/Narsil/silero/resolve/main/silero_vad_16k_op15.onnx
- ブラウザで上記を開くか、次のコマンドで取得できます（PowerShell）:

```powershell
Invoke-WebRequest -Uri "https://huggingface.co/Narsil/silero/resolve/main/silero_vad_16k_op15.onnx" -OutFile "silero_vad_16k_op15.onnx"
```

実行前に `cd public\models` でこのフォルダに移動してください。
